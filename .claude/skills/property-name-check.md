# 物件名チェック

入力された物件名（または物件名のリスト）に対して、以下を実行してください。

## 手順

1. `data/master/properties.csv` を読み込む
2. 入力の物件名を1件ずつ、CSV の `name` 列と **完全一致** で照合する
3. 結果を Markdown の表で返す
   - 列：`入力値`, `判定`, `候補（表記揺れの可能性）`, `物件ID`
   - `判定` は ✅一致 / 🔴不一致 のいずれか
4. 🔴 が1件でもあれば、末尾に「修正候補」をまとめて提示する

## 表記揺れ候補の出し方

完全一致しなかった場合、CSV の `name` 列に対して以下を試して、近い候補を最大3件挙げる。

- 全角／半角の違いを無視した一致
- スペースの有無を無視した一致
- 「号棟」「号室」「棟」などの語尾を取り除いた一致

## 出力例

```
| 入力値                | 判定   | 候補                       | 物件ID |
| --------------------- | ------ | -------------------------- | ------ |
| 東京アパートメント1号棟 | ✅一致 | -                          | P-001  |
| 渋谷スカイハウス      | 🔴不一致 | 渋谷スカイハウス201号室    | -      |
```

## 注意

- マスタに無い物件を勝手に追加しないこと
- 判定結果は必ず人間に確認してもらうこと
